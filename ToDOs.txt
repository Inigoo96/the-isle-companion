# Proyecto: The Isle Companion — estado y TODO
# (Actualizado: 2026-06-25. Para el detalle por sesión ver DOCUMENTACION.txt)

## Qué es
App de escritorio (overlay Electron) para The Isle / Evrima que da info al jugador sin
tocar el juego (solo lee el portapapeles para coordenadas). Convertida en plataforma
multi-tenant: cada servidor de The Isle puede registrarse y la app se adapta a su config.

## Arquitectura (actual)
- Overlay (Electron + JS vanilla, login Steam) -> Backend (Spring Boot + PostgreSQL en
  Railway) -> [Nivel 2, futuro] servidores del juego vía RCON (solo lectura).
- Panel admin (React + Vite en GitHub Pages), identidad 100% Discord.
- Dos identidades SEPARADAS, nunca se mezclan:
    accounts (Steam)  = jugadores del overlay.
    admins   (Discord) = administradores del panel.

## Límites (respétalos)
- La app NUNCA escribe en el juego. Nada de garage de dinos ni comandos del server (eso
  es del bot de Discord de cada server). No parsear Game.ini.
- No persistir el dino actual del jugador (si hace falta, se lee EN VIVO por RCON).
- API devuelve DTOs, nunca entidades JPA. Flyway + ddl-auto=validate (no generar esquema
  con Hibernate; migracion nueva, no editar las aplicadas).
- Coste: gratis/barato (Railway + GitHub Pages). Nada de AWS por defecto.
